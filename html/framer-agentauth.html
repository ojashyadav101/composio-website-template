Step 1 :

Authorize the User to a Toolkit (e.g., GitHub)

Let users securely connect their app accounts (like GitHub) with a single line. Lucy handles the full OAuth flow and waits until the user successfully connects.

Step 2 :

Fetch Prebuilt Tools for the Connected Toolkit

Once connected, Lucy gives you a ready-to-use set of tools for that app — no manual API wiring, just clean, typed functions for all supported actions.

Step 3 :

Power your Agents with Intuition

Pass these tools directly into LLMs (like OpenAI), so your agents can not only think — but also do. With Lucy, function calling is natively integrated.
